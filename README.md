# CCERT - CCTV Installation & Repair Platform 🛡️

A comprehensive platform bridging the gap between customers, technicians, and delivery partners for CCTV products and repair services.

## 🚀 Key Features
- **Customer Portal:** Buy CCTV products, book technicians, and track orders with a real-time Flipkart/Amazon style tracking pipeline.
- **Partner Dashboard:** Real-time assignment acceptance, location routing via Google Maps, and mandatory photo-proof task completion.
- **Admin Workspace:** Full control over product catalog, dynamic technician pricing, order approvals, and profit/expense tracking.
- **Support System:** Integrated ticketing system with full-screen, auto-syncing chat.
- **Offline Reliability:** Built-in offline blockers to ensure data consistency.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend & Database:** Firebase (Firestore, Authentication)
- **Deployment:** Ready for WebView / Android Studio Integration

## ⚙️ How to Run Locally
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file in the root directory and add your Firebase credentials.
4. Run `npm start` to start the development server.
